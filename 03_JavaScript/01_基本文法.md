# JavaScript の基本文法

JavaScript は主に Web ブラウザ上で動作するスクリプト言語。

HTML や CSS と連携し、Web ページに動きや対話性を持たせるために使われる。

サーバーサイド（Node.js など）やモバイルアプリ開発などで使用される。

## JavaScript の文法の特徴的な点をいくつか挙げます。

| 項目     | 説明                                                                                                                                                                       | 具体例                                                            |
| -------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------- |
| 変数宣言 | 値を格納するための箱を準備します。再代入の可否などで使い分けます。                                                                                                         | `const name = "Alice"`; (再代入不可)`let count = 0;` (再代入可能) |
| データ型 | "扱うデータの種類。主要な型には文字列 ("text")、数値 (123)、真偽値 (true/false)、オブジェクト ({key: value})、配列 ([1, 2, 3])、未定義 (undefined)、null などがあります。" | `const obj = { id: 1 };`                                          |
| 関数     | 特定の処理をひとまとめにして再利用可能にしたものです。                                                                                                                     | `function greet(name) { return ""Hello, "" + name; }`             |
| 制御構文 | 処理の流れを制御します。                                                                                                                                                   | `if (...) { ... }` (条件分岐)`for (...) { ... }` (繰り返し)       |
