# CSS によるレイアウトの作り方

## CSS レイアウト

CSS では主に以下の方法でレイアウトを作る。

| 手法      | 特徴                          | 主な用途         |
| --------- | ----------------------------- | ---------------- |
| `display` | 要素の表示形式を変える        | 基本中の基本     |
| `float`   | 要素を左右に回り込ませる      | 簡単な並び替え   |
| `flexbox` | 要素を柔軟に横並び・縦並びに  | モダンな方法     |
| `grid`    | 2 次元（縦 × 横）のレイアウト | 複雑な構成に最適 |

## display プロパティ

基本的な配置方法で、文字や画像の回り込みで使用。

### 基本形：

```css
div {
  display: block;
}
```

| 値             | 意味                           | 例                     |
| -------------- | ------------------------------ | ---------------------- |
| `block`        | 改行して縦に積まれる           | `<div>` , `<p>`など    |
| `inline`       | 横に並ぶ（幅は中身に合わせる） | `<span>`, `<a>`など    |
| `inline-block` | 横に並ぶ＋幅・高さを指定できる | ボタンなど             |
| `none`         | 非表示にする                   | メニュー開閉などに使用 |

## float プロパティ

要素を左・右に回り込ませて配置する。文字や画像の回り込みで使用。

```css
img {
  float: left;
  margin-right: 10px;
}
```

- メリット： テキストと画像を自然に並べられる。
- デメリット： 親要素の高さが崩れやすい（`clear`や`overflow`が必要）。

## flexbox （フレックスボックス）

横並び・縦並び・中央寄せを簡単にでこいる、モダンなレイアウト方法。メニュー・カード一覧など。

### 使い方

```css
.container {
  display: flex;
  justify-content: center; /* 水平方向 */
  align-items: center; /* 垂直方向 */
}
```

| プロパティ        | 説明                           | 例                         |
| ----------------- | ------------------------------ | -------------------------- |
| `display: flex`   | フレックスボックスを有効にする | -                          |
| `flex-direction`  | 並び方向（`row`・`column`）    | 横・縦の切り替え           |
| `justify-content` | 横方向の整列（中央寄せとか）   | `center` / `space-between` |
| `align-items`     | 縦方向の整列                   | `center` / `flex-start`    |
| `gap`             | 要素間の隙間                   | `gap: 16px;`               |

## grid レイアウト

縦・横のマス目で配置を管理する、より強力な方法。ページ全体のレイアウトで使用。

### 主なプロパティ

| プロパティ                 | 内容                          |
| -------------------------- | ----------------------------- |
| `display: grid`            | グリッドを有効にする          |
| `grid-template-columns`    | 列の幅を指定（例：`1fr 2fr`） |
| `grid-template-rows`       | 行の高さを指定                |
| `gap`                      | 行・列の間隔                  |
| `grid-column` / `grid-row` | 各要素の配置を指定            |

## 位置指定 （potion）

要素を自由な場所に配置したいときに使用。固定ヘッダー・重ね配置などで使用。

| 値         | 内容                                   |
| ---------- | -------------------------------------- |
| `static`   | 通常の位置（初期値）                   |
| `relative` | 元の位置を基準に移動                   |
| `absolute` | 親要素を基準に配置                     |
| `fixed`    | 画面に固定（スクロールしても動かない） |
| `sticky`   | スクロールに合わせて固定される         |
