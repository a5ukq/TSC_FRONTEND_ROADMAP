# CSS の基本文法

## class 属性・id 属性の記述方法・役割

- class 属性は、HTML 要素に対して同じスタイルを複数適用したいときに使用する。

```html
<p class="test">...</p>
<p class="test-test2">...</p>
```

複数のクラス名を付ける場合は、半角スペースで区切る。

- id 属性は、HTML 内で一つだけ使う・一意であることを前提にした識別子。

```html
<p id="test">...</p>
```

class 属性とは異なり、同じ HTML 内で同じ id を複数の要素につけるべきではない。

命名規則として、class・id の名前に使用できる文字は「アルファベット・数字、ハイフン、アンダースコア」が基本。先頭に数字やハイフンを使うことは推奨されない。日本語での命名も避ける。

## CSS / HTML 文法の基本構成

- セレクタと宣言ブロックの組み合わせによって、CSS でスタイルを記述

```css
p,
div,
.class_name {
  font-size: 18px;
  font-weight: bold;
}
```

複数のセレクタを使うときは、カンマで区切る。

- ユニバーサルセレクタ「\*」を使うと、全ての要素を対象にできる。

- クラスセレクタと ID セレクタの書き方

  - クラスセレクタ：`.`の後にクラス名を記述。
  - ID セレクタ：`#`の後に id 名を記述。

- インデント・改行のマナーも大事。
  1 行で書くこともできるが、改行して見やすく記述する方が良い。
